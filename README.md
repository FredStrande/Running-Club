# Running Club

A retro-modern running club management game built with Python and Pygame.

## Features

- Main menu with animated title
- Settings screen with volume control
- Background music and sound effects
- Balatro-inspired retro-modern aesthetic

## Requirements

- Python 3.8+
- pygame 2.5.2

## Installation

pip install -r requirements.txt## Running

python main.py## Assets

- Fonts: BoldPixels (included)
- Music: Background track (included)
- Sounds: Button hover effects (included)
